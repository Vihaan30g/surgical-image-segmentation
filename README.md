# Surgical Image Segmentation using Deep Learning

A deep learning project focused on semantic segmentation of laparoscopic surgical scenes using the CholecSeg8k dataset.

## Project Goals

* Build segmentation architectures from scratch
* Handle severe class imbalance
* Experiment with advanced loss functions
* Visualize predictions and feature maps
* Track experiments professionally
* Create a reproducible ML engineering workflow

## Dataset

Dataset used:

* CholecSeg8k

Classes include:

* Organs
* Surgical instruments
* Blood vessels
* Surgical hazards

## Planned Features

* Custom U-Net implementation
* Attention-based segmentation improvements
* Dice / IoU metrics
* TensorBoard monitoring
* Experiment tracking
* Visualization tools
* Inference pipeline
